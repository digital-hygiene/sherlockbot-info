# Sherlock Bot — справочная информация

Sherlock Bot — это независимый справочный ресурс, посвящённый цифровой гигиене, безопасности в Telegram и проверке неизвестных контактов.

Проект создан для повышения осознанности пользователей при взаимодействии с:
- неизвестными номерами
- подозрительными аккаунтами в Telegram
- возможными сценариями спама и мошенничества

## Что такое Sherlock Bot

Sherlock Bot — Telegram-бот, который помогает пользователям:
- оценивать потенциальные риски
- понимать контекст сообщений и звонков
- принимать более безопасные решения

Сервис носит **справочный характер** и не является инструментом правоохранительных или коммерческих проверок.

## Основные возможности
- Проверка неизвестных номеров и контактов
- Анализ репутационных сигналов
- Рекомендации по цифровой безопасности
- Подсказки по настройкам приватности в Telegram
